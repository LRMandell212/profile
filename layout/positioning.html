Exploring Positioning

Positioning

One way to disrupt normal flow is with the CSS position property.

The position property takes one of five single keywords: static, relative, absolute, fixed, or sticky. By default all elements are static (normal flow). Once an elenent has been given a position property value other than static it is defined as being "positioned".

  • Relatively positioned elements are offset a given amount from their normal position within the document, but without the offset affecting other elements.
  • The absolutely positioned element is positioned relative to its nearest positioned ancestor (i.e., the nearest ancestor that is not static). An element that is absolutely positioned is taken out of the flow; thus, other elements are positioned as if it did not exist.
  • Fixed positioning is similar to absolute positioning, with the exception that the element's containing block is the initial containing block established by the viewport. This can be used to create a "floating" element that stays in the same position regardless of scrolling.
  • A stickly positioned element is treated as relatively positioned until it crosses a specified threshold, at which point it is treated as fixed until it reaches the boundary of its parent.
