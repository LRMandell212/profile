Exploring Normal Document Flow

Elements on a webpage lay out in the normal flow, if you have not applied any CSS to change the way they behave. Remember there are basically two types of elements—block and inline.

A block-level element occupies the entire horizontal space of its parent element (container), and vertical space equal to the height of its contents, thereby creating a "block". The most important implication of this is that (in most browsers) anything after a block element will start on a new line.

In contrast to block-level elements an inline-level elements is only as wide as its content requires. The most important implication of this is that inline elements do NOT start on a new line, they continue in the normal flow of the content. Generally inline elements can contain only other inline elements.

With CSS you can change how elements behave either by adjusting their position in that normal flow, or removing them from it altogether.
